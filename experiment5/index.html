Description

A generator that reads an image and renders it using three basic shapes including circles, rectangles, and triangles while also making adjustable incremental changes.

Technical

Images get initialized by initDesign() as well as design parameters. These design parameters are then used by renderDesign() to determine which shape, coordinates, and color it should use to render the image. The shapes are mutated with mutateDesign() which uses a mut() function consisting of constrain() and randomGaussian() to make the shapes move incrementally.

Reflection

I really like how my experiment turned out. Although there are many different shapes being created, it still retains the original image and adds some nice contrast with the shapes clashing with one another. For some reason I had difficulty getting the colors for the shapes which is what took me the longest to do in this experiment. If I were to add improvements to my experiment, I would like to add other sliders for the user to adjust the image further. One idea I had was having an opacity slider which changes the opacity of the shapes being created, which could help create more unique images.
